## Prereqs
* [Monafont](https://github.com/githubnext/monaspace) 
* [Wezterm](https://wezfurlong.org/wezterm/index.html)

## Setup

1. clone in home dir
2. cd into dotfiles
3. run `./setup.sh`
4. Reload terminal
5. Go into tmux and ctrl + leader + I to apply tmux changes
